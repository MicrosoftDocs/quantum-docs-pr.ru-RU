Возвращает систему генератора, зависящую от времени, в соответствии с Хамилтониан `H(s) = 0` .

```qsharp
function IdentityTimeDependentGeneratorSystem () : Microsoft.Quantum.Simulation.TimeDependentGeneratorSystem
```
