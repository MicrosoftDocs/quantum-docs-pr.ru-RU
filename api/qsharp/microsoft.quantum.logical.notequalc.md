# <a name="notequalc-function"></a>Функция Нотекуалк

Пространство имен: [Microsoft. тактов. Logical](xref:Microsoft.Quantum.Logical)

Пакет: [Microsoft. такт. Standard](https://nuget.org/packages/Microsoft.Quantum.Standard)


Возвращает значение true только в том случае, если два входных значения не равны.

```qsharp
function NotEqualC (a : Microsoft.Quantum.Math.Complex, b : Microsoft.Quantum.Math.Complex) : Bool
```


## <a name="input"></a>Входные данные

### <a name="a--complex"></a>а. [сложный](xref:Microsoft.Quantum.Math.Complex)

Первое сравниваемое значение.


### <a name="b--complex"></a>б. [сложные](xref:Microsoft.Quantum.Math.Complex)

Второе сравниваемое значение.



## <a name="output--bool"></a>Выходные данные: [bool](xref:microsoft.quantum.lang-ref.bool)

`true` Если и, только если `a` не равно `b` .
